# DECISIONS

## 1. ローカルファースト設計

- すべてのデータは端末内に保存し、クラウド同期は行わない。
- プライバシーを重視し、外部通信を不要とする方針。

## 2. 写真は写真ライブラリに保存

- 画像データはアプリ内ストレージではなく、iOS 写真ライブラリに保存する。
- `localIdentifier` を使って画像参照を管理することで、ストレージ容量や PhotoKit の利便性を活かす。

## 3. JSON ファイルを永続化に採用

- `LabelStore` で `Application Support` に JSON ファイル保存を採用。
- データベース導入は不要なため、シンプルな構造を優先。
- 保存形式はラッパーオブジェクトを持たない `[LabelEntry]` の配列そのものとする。

## 4. SwiftUI を UI フレームワークに選択

- 既存のコードベースが SwiftUI で構築されているため、継続して SwiftUI を使用。
- `NavigationStack` / `sheet` を使い、画面遷移をシンプルに保つ。

## 5. サービス層を明確に分離

- `LabelStore` と `PhotoLibraryService` を責務別に分離する。
- Views はデータ操作を直接行わず、サービス経由で操作する。

## 6. テスト戦略は徐々に追加

- まずは動作確認と QA を優先し、必要に応じてユニットテストを追加する。
- 画像保存などの外部依存部分はモック化してテスト可能にする。

## 7. 新規追加は `+` から統一

- 一覧の右上 `+` から新規追加を開始する導線に統一する。
- 画像追加は編集画面内で `カメラ` と `フォトライブラリ` の両方を選べるようにする。

## 8. 詳細画面は参照専用 + 明示的編集

- 詳細画面は表示専用とし、`編集` ボタンで編集画面に遷移する。
- 削除は編集画面下部に集約し、詳細画面からは直接削除させない。

## 9. カメラ撮影画像はメタデータ保持で保存

- カメラ撮影した画像は `UIImage` の再エンコード時に EXIF / TIFF メタデータを埋め戻して写真ライブラリへ保存する。
- フォトライブラリから選択した既存写真は再保存せず、既存 `PHAsset` の `localIdentifier` をそのまま参照する。
- 保存したメタデータは UI へ表示せず、写真ライブラリ上の原本保持を優先する。
- 一方で表示継続性のため、登録時にはアプリ内に縮小バックアップ画像も保持し、写真ライブラリから読めない場合のフォールバックに使う。

## 10. 日時は `登録日時` に統一

- エントリは `createdAt` / `updatedAt` を個別管理せず、単一の `registeredAt` を保持する。
- カメラ撮影からの登録では、撮影画像メタデータ上の撮影時刻を `registeredAt` に採用する。
- フォトライブラリからの登録では、画像の EXIF 撮影時刻を優先し、取得できない場合のみ `PHAsset.creationDate` を使う。
- 既存データ読込時は、旧 `createdAt` / `updatedAt` から `registeredAt` へ後方互換で移行する。

## 11. 起動直後にアプリ内スプラッシュを表示

- iOS 標準の Launch Screen の直後に、アプリ内で短時間のスプラッシュ画面を表示してから一覧画面へ遷移する。
- スプラッシュ画面はアプリ本体の初期化を妨げない単純な SwiftUI View とし、ユーザー入力は受け付けない。
- スプラッシュ画面は設定済みのアプリアイコン画像とアプリ名を中央に表示し、短時間のフェードアウトで一覧画面へ遷移する。

## 12. 補助導線は一覧左上メニューへ集約

- 一覧画面の主タイトルはナビゲーション中央に寄せ、ボトルアイコンとアプリ名を一体で見せる。
- `設定` と `その他` の補助導線は、一覧画面左上のハンバーガーメニューに集約する。
- ハンバーガーメニューは即時選択メニューではなく、左から現れるドロワーとして表示する。
- `その他` 内に `利用規約` と `プライバシーポリシー` の静的文書画面を配置する。
- 一覧画面は大タイトル表示を使わず、フィルターとの間に不要な上余白を作らない。

## 13. ユーザー向けバージョンはセマンティック表記

- 設定画面に表示するアプリバージョンは `major.minor.patch` 形式のユーザー向けバージョンのみとする。
- `CFBundleVersion` は内部ビルド番号として保持し、設定画面には表示しない。
